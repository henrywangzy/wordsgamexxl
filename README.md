# 单词消消乐 - 移动端H5版

📚 一款专为移动端设计的英语单词学习H5小游戏

## 🎮 游戏简介

通过趣味配对游戏，让学习英语单词变得更加有趣！专为9:16竖屏优化，支持六种年级难度，是初中生学习沪教版Unit 5英语词汇的绝佳工具。

## 🚀 快速开始

### 在线体验
直接在浏览器中打开 `index.html` 即可体验游戏。

### 本地部署
1. 下载所有文件到本地目录
2. 使用任意Web服务器托管
3. 访问 `index.html`

推荐使用Python简单服务器：
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后访问: `http://localhost:8000`

## 🎮 游戏玩法

1. **选择年级**: 在首页选择合适的年级难度
2. **开始游戏**: 点击年级按钮直接进入游戏
3. **匹配单词**: 点击英文单词和对应的中文释义进行配对
4. **获得分数**: 成功配对获得分数，连续配对有连击奖励
5. **完成挑战**: 在时间内消除所有配对或等待时间结束
6. **查看成绩**: 游戏结束后查看得分、正确率等统计信息
7. **分享成就**: 可以分享成绩到社交媒体

## 🔧 自定义开发

### 添加新单词
编辑 `js/word-database.js` 文件，在 `WordDatabase` 数组中添加新单词：

```javascript
{ english: 'newWord', chinese: '新单词', emoji: '📝', partOfSpeech: 'n.' }
```

### 修改难度设置
编辑 `js/word-database.js` 文件中的 `GradeConfig` 对象：

```javascript
const GradeConfig = {
    'grade1': { time: 120, level: 'easy', pairs: 18, difficulty: 1.0 },
    // ... 其他年级
};
```

### 自定义样式
编辑 `css/style.css` 文件修改游戏外观。
